# `docs/spec.md`

## Visione

Applicazione mobile per studenti universitari che aiuta a gestire lezioni, materiali, scadenze e promemoria in modo centralizzato.

## Funzionalità MVP

- registrazione e login;
- dashboard personale;
- calendario esami;
- download materiali da backend remoto;
- notifiche push per scadenze;
- cache offline dei dati principali.

## Vincoli tecnici

- `.NET MAUI` con MVVM;
- API REST remota;
- database locale per uso offline;
- notifiche locali e push.

## Obiettivi futuri

- chat tra studenti;
- integrazione AI per suggerimenti di studio.
